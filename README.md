# 🌿 Eco Dry Cleaners — Web App

A full-featured React web application for an eco-friendly laundry service business.

---

## 🚀 Getting Started

### Install & Run

```bash
npm install
npm start
# Opens at http://localhost:3000
```

## 🗂️ Pages

| Route | Description |
|-------|-------------|
| `/` | Home / Landing Page |
| `/login` | Customer Login |
| `/register` | New Customer Registration |
| `/dashboard` | Customer Dashboard |
| `/booking` | Multi-step Booking Form |
| `/tracking` | Order Tracking by ID |
| `/payment/:id` | M-Pesa Payment |
| `/services` | Services Listing |
| `/contact` | Contact Form |
| `/profile` | Profile Settings |
| `/admin` | Admin Panel (admin only) |

---

## 🛠️ Tech Stack

- **React 18** + **React Router v6**
- **Recharts** — Admin analytics charts
- **Lucide React** — Icons
- **localStorage** — Data persistence (no backend needed)

---

## 📦 Deploy (Production Build)

The `build/` folder is ready to deploy on Netlify, Vercel, Firebase, or GitHub Pages.

```bash
npm run build
```
